Frequently asked questions

Is it safe to use?

Absolutely. The extension only runs on Salesforce domains and doesn't collect or store any of your org data. All operations use Salesforce's native APIs.

Does it work with all Salesforce editions?

Yes, the extension works with all Salesforce editions including Professional, Enterprise, Unlimited, and Developer editions. It also supports both Classic and Lightning Experience.

Is there a free version?

Yes! The core features are completely free. We also offer a Pro version with advanced features like bulk operations and custom shortcuts for power users.

How often is it updated?

We release updates every 2-3 weeks with new features and improvements. The extension auto-updates, so you always have the latest version.
